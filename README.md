# vue-terminal
`vue-terminal` is a terminal emulator written in Vue. It is currently being used to power my [portfolio site](https://jsmith.github.io).

## Project setup
```
npm i
```

### Startup the Development Environment (Storybook)
```
npm run serve:storybook
```
